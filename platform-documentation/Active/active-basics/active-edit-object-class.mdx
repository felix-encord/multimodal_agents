---
title: "Bulk Edit Object Class"
description: "Learn how change the labels on data units in Collections."
---

Active supports bulk changes made to the object label classes on data units in **Label Collections** to an Annotate Project.

<Note>Active Label Collections support editing object label classes of one object label type to the same label type. For example, bounding box to bounding box or bitmask to bitmask.</Note>

<Tip>Use the **Display** settings to help find labels that require action.</Tip>

![Labels Display settings](https://storage.googleapis.com/docs-media.encord.com/static/img/active/active-collections-labels-display.gif)

**To update object labels using a Label Collection:**

1. Log in to the Encord platform.
   The landing page for the Encord platform appears.

2. Click **Active** in the main menu.
   The landing page for Active appears.

3. Click the Project.
   The landing page for the Project appears with the _Explorer_ tab selected.

4. Select **Labels** on the Explorer page.

5. Create one or more Collection by searching, sorting, and filtering until you have the subset of the data you need.
  
   ![Create Label Collection](https://storage.googleapis.com/docs-media.encord.com/static/img/active/active-collections-labels-edit-labels-create-collection.gif)

6. Click **Collections**.
   The Collections page appears.

   ![Bulk Label Update](https://storage.googleapis.com/docs-media.encord.com/static/img/active/active-collections-label-collection-bulk-label-update.gif)

7. Select one or more **Label Collections** from the Collection list.

8. Click **Bulk Actions**.
   A menu appears.

9. Select **Edit Object Class**.
   A dialog appears.

10. Select the object label classes you want to change under **Classes to change**.

11. Select the object label class you want to change to under **Change to**.

12. Click **Initiate Bulk Edit Object Class**.
    The process begins. A success messages once the action completes.
