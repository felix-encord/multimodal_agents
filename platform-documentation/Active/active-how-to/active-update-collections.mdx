Once you have created a Collection, you are likely going to need to update the Collection as more data is added and as you find more interesting samples that can help you improve your model's performance.

<AccordionGroup>
   <Accordion title="To update an existing Collection:">

1. Log in to the Encord platform.
   The landing page for the Encord platform appears.
   
2. Click **Active** in the main menu.
   The landing page for Active appears.

3. Click the Project.
   The landing page for the Project appears with the _Explorer_ tab selected.

4. Search, sort, and filter your data/labels/predictions until you have the subset of the data you need.

5. Select one or more of the images.
   A ribbon appears at the top of the Explorer workspace.

6. Click **Select all** to select all the images in the subset.

7. Click **Existing Collection**.

8. Select one or more Collections to add the selected images to.

9. Click **Submit**.

  </Accordion>
</AccordionGroup>
