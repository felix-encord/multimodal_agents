## Label Editor Document Support

The Label Editor supports the following document file types:
- PDF (.pdf)

### Document Label Support

Documents support the following label types:

- Bounding boxes
- Polygons
- Bitmasks
- Classifications
- Text Region

<Tip>We recommend adding attributes to your label types when creating your Ontology for labeling documents.</Tip>

<Tip>The color you specify for each object label in your Ontology can make a big difference in the ease and effectiveness of labeling items. For example, lighter colors are more difficult to see on a white background.</Tip>

## Label Documents

**Label display** settings can help provide a better labeling experience for each annotator.

![Label Display Settings](https://storage.googleapis.com/docs-media.encord.com/static/img/annotate/label_editor/annotate-label-display-settings.gif)

<Tip>Press and hold the <kbd>SHIFT</kbd> key while using your mouse/trackpad to move around each page of a document.</Tip>

### Label PDFs

Annotate supports the following label types for PDFs:

- Bounding boxes
- Polygons
- Bitmasks
- Classifications

<Note>Classifications support being applied on a range of pages in a PDF.</Note>

Use **Show text outline** button feature when labeling documents to streamline your annotation process. When used with object label hotkeys, labeling becomes a smooth efficient process.

![Show text outline](https://storage.googleapis.com/docs-media.encord.com/static/img/annotate/label_editor/annotate-label-show-text-outlines-basic.gif)

#### Bounding Boxes

1. Click the **Show text outline** button to enable the feature.
2. Press the hotkey for the bounding box, and the label appears around the area you highlight.

<Tip>Bounding boxes are the quickest way to label documents in the Label Editor.</Tip>

![Label docs with Bounding Box](https://storage.googleapis.com/docs-media.encord.com/static/img/annotate/label_editor/annotate-label-documents-bounding-box.gif)

#### Bitmasks

1. Click the **Show text outline** button to enable the feature.
2. Press the hotkey for the bitmask, and then start labeling.

![Label documents with Bitmask](https://storage.googleapis.com/docs-media.encord.com/static/img/annotate/label_editor/annotate-label-documents-bitmask.gif)


#### Polygons

1. Click the **Show text outline** button to enable the feature
2. Press the hotkey for the polygon, and then start labeling.

![Label documents with Polygon](https://storage.googleapis.com/docs-media.encord.com/static/img/annotate/label_editor/annotate-label-documents-polygons.gif)
