Your data is registered in the _Files_ section of _Index_ where it is organized into folders and sub-folders. Importing your data into Encord is a multi-step process: 

1. Create an Azure integration.  
2. Set up an Azure integration.
3. Create a JSON or CSV.
4. Create a folder to store your data in Encord.
5. Register your data to the folder. 

<Tip>See our [Azure integration documentation](/platform-documentation/General/annotate-data-integrations/annotate-azure-blob-integration) for a detailed guide to setting up an integration.</Tip>

## Step 5: Create a Folder

1. Navigate to _Files_ section of _Index_ in the Encord platform.
2. Click **+ New folder**.
A dialog to create a new folder appears.


<img src="https://storage.googleapis.com/docs-media.encord.com/static/img/storage-new-folder.png" width="600"/>


3. Give the folder a meaningful name and description. 
4. Click **Create** to create the folder. 
The folder is listed in _Files_.
