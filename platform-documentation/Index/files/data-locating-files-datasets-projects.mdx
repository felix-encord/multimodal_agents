_Files_ allows you to easily locate data, and learn how it is being used across Datasets and Projects in Encord. 

**To see which Datasets a data unit has been added to**:

1. Navigate to, and select the data unit you want to locate in _Files_.

<Tip>Use the search bar to search for a specific file.  </Tip>

2. Once the data unit is selected, click the _Linked to_ tab on the right of the screen. 
The total number of Datasets the file is linked to, along with their names is shown. 

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/storage-linked-datasets.png" width="600"/>
</div>

**To see which Projects a Dataset is attached to**:

3. Click the ellipses icon next to the name of a Dataset. 

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/view-dataset-storage.png" width="600"/>
</div>

4. Select _View dataset_. 
A view of the Dataset appears. 

5. Navigate to the _Settings_ tab of the Dataset.

6. Select _Linked projects_ to see a list of all Projects the Dataset is attached to. 

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/dataset-linked-projects.png" width="600"/>
</div>
