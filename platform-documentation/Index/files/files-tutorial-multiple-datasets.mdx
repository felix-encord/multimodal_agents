Data stored in _Files_ can be added to one, or multiple Datasets with ease. 

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/add-data-multiple-datasets.gif" width="650" height="280" />
</div>


1. Navigate to the folder containing the data you want to add to multiple Datasets. 

2. Select the data you want to add to the Datasets. 

<Tip>To add multiple files to a Dataset, bulk-select files by selecting the first file, and then selecting the last file while holding the <kbd>Shift</kbd> key on your keyboard.</Tip>

3. Right-click your selection and select **Add to dataset**. 
A dialog appears. 

4. Add data to a Dataset by clicking **Add** next to a Dataset's name. Add data to multiple Datasets by clicking **Add** next to each Dataset you want to add the data to. 

<Note>You cannot add data to [mirrored Datasets](/platform-documentation/Annotate/annotate-datasets/annotate-datasets#mirrored-datasets). </Note>
