Any Index Collection can be exported to CSV. The CSV file contains the following:

- File ID: Specifies the unique data ID for each image, image group, image sequence, or video in the Collection.

- File Name: Specifies the file name for the image or video in the Collection.

- Collection Hash: Specifies the unique ID for the Collection.

**To export a Collection to CSV:**

1. Log in to the Encord platform.
   The landing page for the Encord platform appears.

2. Go to **Index > Files**.
   The _All folders_ page appears with a list of all folders in Encord.

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/Index/index-001.png" width="600" />
</div>

3. Click in to a folder.
   The landing page for the folder appears and the _Explorer_ button is enabled.

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/Index/index-002.png" width="600" />
</div>

4. Click the **Explorer** button.
   The _Index Explorer_ page appears.

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/Index/index-03.png" width="600" />
</div>

5. Click **Collections**.
   The _Collections_ page appears.

<div class="flex justify-center">
    <img src="https://storage.googleapis.com/docs-media.encord.com/static/img/Index/index-export-to-csv.png" width="600" />
</div>

6. Select the checkbox for the Index Collection for export.

7. Click the more icon.
   A small menu appears.

8. Click **Generate CSV** from the menu.
   The _Download Generated CSV_ dialog appears when Active finishes generating a CSV file for the Collection.

9. Click **Download CSV**.
   The CSV file downloads to your local computer.
